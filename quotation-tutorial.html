HTML Quotation

In HTML, quotation elements are used to define and display quotations or pieces of text that are taken from another source.

There are varying HTML Quotation Elements:

1. < blockquote>

The HTML < blockquote> element defines a section that is quoted from another source. It is typically used to indicate that a section of text is a direct quotation from another source. The < blockquote> element not only helps in formatting quotations but also provides significant meaning to your content, making it clear that a particular section of text is a quotation.

2. < q>

The HTML < q> element defines defines a short quotation. It is a way to mark up quotations.It helps browsers and search engines recognize and style quoted text appropriately.

3. < abbr>

The HTML < abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". It's typically used to provide a brief explanation or expansion of an abbreviation. A significant way to mark up abbreviations and acronyms in your HTML documents, helping assistive technologies and search engines to better understand the content.

4. < address>

The HTML < address> tag defines the contact information for the author/owner of a document or an article. It used to include details such as the author's name, email address, physical address, phone number, or any other contact information. It also provides significant information to browsers and search engines that this is contact information. This element is often seen in the context of articles, blog posts, or documents to specify the author's contact information. It should not be used to wrap general contact details in a webpage's footer, for example, which should be done with a different element, like a < footer>.

5. < cite>

HTML < cite> for Work Title defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.). The text in the < cite> element usually renders in italic. It's a way to provide proper attribution and emphasize the title of the work within your content. This element is useful in academic or bibliographic contexts to mark up citations accurately.

Note: the < cite> element is for marking up titles and should not be confused with the < blockquote> element, which is used for longer quotations.

6. < bdo>

HTML for Bi-Directional Override. The HTML tag is used to override the current text direction. It is used to control the direction of text within an HTML element. It is primarily used in cases where you need to change the text direction within a specific part of the content to handle languages or script. This element is also used for handling bidirectional text in HTML, and it should be used with caution, typically in situations where the default text direction is not sufficient to properly display the content.
