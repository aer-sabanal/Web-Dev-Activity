HTML Styles

Styles in HTML are basically rules that describe how a document will be presented in a browser. Styles in HTML are basically rules that describe how a document will be presented in a browser.

There are 3 ways of implementing style in HTML:
Inline Style: In this method, the style attribute is used inside the HTML start tag.
Embedded Style: In this method, the style element is used inside the element of the document.
External Style: In this method, the style element is used to point to an external CSS file.

HTML Style Example

Example: Result:
<!DOCTYPE html>
<html>
<head>
<style>
h1 {color:blue }
p {color:red }
</style>
</head>
<body>
<h1> This is a heading </h1>
<p> This is a paragraph</style>
</body>
</html>
