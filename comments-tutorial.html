HTML Comment

An HTML comment tag is a way to insert comments into the source code of a webpage. It doesn't have any impact on the display of the page in the browser; it's purely for developers to include notes or explanations within the HTML code.

You can add comments to you HTML Source by using the following syntax:

< !-- Write your comments here -- >

Note: Notice that there is an exclamation point (!) in the start tag, but not in the end of tag. Anything between < !-- and --> is treated as a comment and is ignored by the browser when rendering the webpage.

1. Add Comments

With comments you can place notifications and reminders in you HTML Code:

Example:

< !-- This is a comment about the following paragraph -->
< p> This is a sample paragprah on my webpage. < /p>
< !-- Feel free to add more comments as needed -->
