HTML <code> For Computer Code

The <code> element is an HTML tag used to define a piece of computer code. It is typically used to display code snippets within a web page, often for educational or reference purposes. The content inside the <code> tag is displayed in the browser's default monospace font, which is a font designed for displaying text with a consistent width for each character.

In the given example, the code snippet x = 5; y = 6; z = x + y; print(z) is enclosed within the <code> tag.
This ensures that the code is displayed in the browser's monospace font and with the proper indentation.

<code>
x = 5;
y = 6;
z = x + y;
print(z)
</code>

Result:
When you view this code in a web browser, you will see the following output:
x = 5;
y = 6;
z = x + y;
print(z)
