HTML Images

HTML images are like pictures in a coloring book. They make a webpage pretty and fun.
Images can improve the design and the appearance of a web page.

HTML Image Syntax:

The HTML <img> tag is used to embed an image in a web page.

Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

The <img> tag is empty, it contains attributes only, and does not have a closing tag.
The <img> tag has two required attributes:
src - Specifies the path to the image
alt - Specifies an alternate text for the image

<img src="url" alt="alternatetext">

Adding an Image:

In HTML, we use a special code called to add an image to a webpage. It's like telling the computer, "Put this picture here." Here's an example:

<img src="kitten.jpg" alt="A cute kitten">

<img> is like a magic spell to show a picture.
src="kitten.jpg" is the name of the picture we want to show.
alt="A cute kitten" is a message in case the picture can't be seen, telling us what the picture is about.

Image Size:

You can also tell the computer how big the picture should be. It's like saying, "Make it this tall and this wide." We use the width and height tricks:

<img src="puppy.jpg" alt="An adorable puppy" width="200" height="150">

width="200" says how wide the picture should be.
height="150" says how tall the picture should be.

Basically, HTML helps us put beautiful pictures on web pages. It's like using magic to decorate a page with your favorite coloring book pictures
