The Target Attribute

The target attribute in HTML is used to specify where to open a linked document. It is typically used with the <a> and <area> elements to control how hyperlinks behave.
For example, the following code will open the linked document in a new tab:

<a href="https://www.example.com"  target="_blank"<Example Link</a>>

The target attribute can have the following values:

_blank: Opens the linked document in a new tab or window.

_self: Opens the linked document in the current window or tab.

_parent: Opens the linked document in the parent frame.

_top: Opens the linked document in the full browser window, replacing all current frames.

_framename: Opens the linked document in the frame with the specified name.
