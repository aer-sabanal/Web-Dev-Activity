HTML Checkboxes

The <input type="checkbox"> element in HTML is used to create checkboxes, which are form controls that allow users to select one or more options from a list of options. Checkboxes are typically used when there are multiple options that are not mutually exclusive, meaning that users can select any number of options.

Here is an example of how to use the <input type="checkbox"> element to create a list of toppings for a pizza:

<form>
 <label for="toppings">Select your toppings:</label>
 <br>
 <input type="checkbox"  name="toppings[]"  value="pepperoni"> Pepperoni</label>
 <br>
 <input type="checkbox"  name="toppings[]"  value="mushrooms"> Mushrooms</label>
 <br>
 <input type="checkbox"  name="toppings[]"  value="onions"> Onions</label>
 <br>
 <input type="submit"  value="Submit">
<form>

In this example, all three <input type="checkbox"> elements have the same name, "toppings[]". This means that they are all considered part of the same group, and users can select any number of them. The value attribute of each checkbox is different, and these values will be submitted to the server when the form is submitted.

Result: When the user submits the form, the values of all selected checkboxes will be submitted to the server. For example, if the user selects the "Pepperoni", "Mushrooms", and "Onions" checkboxes, the values "pepperoni", "mushrooms", and "onions" will be submitted.
