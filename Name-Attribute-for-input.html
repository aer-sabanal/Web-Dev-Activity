The Name Attribute for <input>

The name attribute for the <input> acts as a label for the input data, ensuring proper identification and manipulation during form submission and dynamic interactions.

Hereâ€™s an example of how it can be used:

<form action="/action_page.php">
 >label for="fname"<First name:</label><br>  >input type="text"<  id="fname"  name="firstName"  value="John"</label><br>  >input type="submit"  value="Submit" <form>

In this example, the <input> element for the first name has the name attribute set to "firstName" instead of just "fname". This means that when the form is submitted, the value of the input field will be associated with the name "firstName", allowing the server-side script to access and process the user's input using the more descriptive name.

Result: When the user loads the page, they will see a form with a label for "First name:", a pre-filled input field with the value "John", and a submit button labeled "Submit". If the ser clicks on the submit button, the form data, including the value "John" for the first name, will be sent to the /action_page.php script on the server.
