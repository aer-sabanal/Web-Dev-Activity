HTML CSS

Cascading Style Sheets (CSS) is used to format the layout of a webpage. If HTML is like the bread of the sandwich, CSS is like the special sauce and toppings on the sandwich that make it look pretty and unique.

3 Ways to add CSS to html document

Inline: by using the <style> attribute inside HTML elements
Internal: by using the <style> element in <head> section.
External: by using a <link> element to link to an external CSS file.

Inline CSS

An inline CSS is used to apply a unique style to a single HTML element.

Note: Imagine you have a coloring book. In this book, you can color each picture differently using your crayons. Inline CSS is like using a crayon on each picture. You can change the colors or make things bold or italic right there on the picture itself. It's like giving specific instructions to each picture in your coloring book.

Example:

<!DOCTYPE html> <html>
<body>
<h1 style="color: black;">A Blue Heading</h1>
<p style="color:blue;">A red paragraph.</p>
</body>
</html>

Internal CSS

An internal CSS is used to define a style for a single HTML page.
An internal CSS is defined in the <head> section of an HTML page, within a <style> element.

Note: Now, think of your school's dress code. It tells everyone in your school how they should dress. Internal CSS is similar. It's like having a page in your coloring book that tells all the pictures how they should look. So, instead of changing each picture individually, you tell a whole page to look a certain way.

Example:

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}
</style>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p> </body>
</html>

External CSS

An external style sheet is used to define the style for many HTML pages.

Note: Sometimes, schools have uniforms. The uniform is like a set of clothes that all the students wear. External CSS is similar. It's like having a special box of crayons (a separate file) that you and your friends can all use to color your coloring books. This box of crayons can be shared among many coloring books (web pages). So, you only need to change the crayons in one box to change how all the coloring books look.

To use an external style sheet, add a link to it in the <head> section of each HTML page:
External style sheets can be referenced with a full URL or with a path relative to the current web page.

<link rel="stylesheet" href="/html/styles.css">

Example:

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="styles.css"> </head> <body> <h1>This is a heading</h1>
<p>This is a paragraph.</p> </body>
</html>



To sum it up, inline is like coloring each picture individually, internal is like having a page of instructions for your coloring book, and external is like having a shared set of special crayons for many coloring books. They all help make web pages look pretty and neat!

Common CSS properties:

The CSS color property defines the text color to be used.
The CSS font-family property defines the font to be used.
The CSS font-size property defines the text size to be used.
The CSS border property defines a border around an HTML element.
The CSS padding property defines a padding (space) between the text and the border.
The CSS margin property defines a margin (space) outside the border.
