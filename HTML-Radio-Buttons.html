Radio Buttons

The <input type="radio"> element in HTML is used to create radio buttons, which are form controls that allow users to select one option from a group of related options. Radio buttons are typically used when there are a limited number of mutually exclusive choices.

Hereâ€™s an example of how it can be used:

<form>
 <label>Favorite Color:</label>
 <br>  <label><input type="radio"  name="color"  value="red"> Red</label>
 <label><input type="radio"  name="color"  value="green"> Green</label>
 <br>  <label><input type="radio"  name="color"  value="blue"> Blue</label>
 <label><input type="submit"  value="Submit">  </form>

In this example, the three <input type="radio"> elements all have the same name, "color". This means that they are all part of the same group, and only one of them can be selected at a time. The value attribute of each radio button is different, and this value will be submitted to the server when the form is submitted.

Result: When the user submits the form, the value of the selected radio button will be submitted to the server. For example, if the user selects the "Red" radio button, the value "red" will be submitted.
