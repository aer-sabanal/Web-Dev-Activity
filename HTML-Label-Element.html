The <label> Element

The <label> element is used to associate a text label with a form control. It provides a descriptive caption for the form control, enhancing user understanding and accessibility.

</label>

<label for="email">Email Address:</label>
<input type="email">id="email"   name="email"   required>

Result: When the user clicks on the label text, the corresponding form control will be focused, allowing the user to easily enter their input. The label also provides clear instructions for each input field, improving the user experience and accessibility.
