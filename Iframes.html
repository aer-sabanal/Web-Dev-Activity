HTML Iframes

HTML Iframes, also known as Inline Frames, are used to display a nested webpage (a webpage within a webpage). The HTML <iframe> tag defines an inline frame. An inline frame is used to embed another document within the current HTML document.

Here’s an example of how it can be used:

<iframe src="https://www.example.com" title="Example Website" height="500" width="500"></iframe>

In this example, the src attribute is used to specify the URL of the website to be loaded. The title attribute is included for screen readers to read the title to users. The height and width attributes are used to set the size of the iframe.

You can also use CSS to set the width and height of the iframe using the style attribute. For example:

<iframe src="https://www.example.com" style="height:200px;width:300px" title="Example Website"></iframe>

It’s important to add height and width to allocate space on the webpage for the iframe. It prevents content from moving when the iframe loads.
