HTML Attributes

HTML attributes are always placed inside the opening tag of an element. The opening tag is the part of the HTML code that tells the browser what kind of element it is dealing with. For example, the opening tag for a paragraph element is <p>.

To add an attribute to an element, you simply write the attribute name, followed by an equal sign (=) and then the attribute value. For example, the following HTML code creates a paragraph element with a red font color:

<p color:style="color: blue;">This is a paragraph with a blue font color. </p>

In this example, the style attribute is used to set the font color of the paragraph to blue.

Some HTML attributes have predefined values. For example, the href attribute of an <a> element (anchor element) is used to specify the destination of a link. The predefined value for the href attribute is the empty string. This means that if you do not specify a value for the href attribute, the link will not go anywhere.

Other HTML attributes do not have predefined values. For example, the alt attribute of an <img> element (image element) is used to provide a text description of the image. The alt attribute is required for accessibility reasons, so that people who cannot see the image can still understand what it is.

You can add multiple attributes to an element by separating them with spaces. For example, the following HTML code creates a heading element with a red font color and a center alignment:

<h1 style="color: blue; text-align: center;">This is a heading</h1>

In this example, the style attribute is used to set the font color and alignment of the heading to blue and center, respectively.

HTML attributes are a very powerful tool that can be used to customize the appearance and behavior of HTML elements.
